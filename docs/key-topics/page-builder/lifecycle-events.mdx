## Overview

In our Page Builder Editor (hereinafter: Editor), there is a number of React hooks which can make building custom elements a lot easier. In this article we cover some of the more useful hooks that you might use quite often, if you're developing custom elements.

## Hooks

### useActiveElement

This hook provides access to the active (selected) element, and all of its properties. Use this hook when you need to access element's data, settings, etc.

#### How to use this hook?

```typescript
import { useActiveElement } from "@webiny/app-page-builder/editor/hooks/useActiveElement";

const MyComponent = () => {
  const element = useActiveElement();
};
```

### useActiveElementId

This hook provides access to the active (selected) element ID. The difference between this and the previous hook is that this one _only_ returns the ID, which is a simple string, and it's a lot cheaper and faster to execute. Use this hook if you only need to know the ID of the active element.

#### How to use this hook?

```typescript
import { useActiveElementId } from "@webiny/app-page-builder/editor/hooks/useActiveElement";

const MyComponent = () => {
  const elementId = useActiveElementId();
};
```

### useDisplayMode

This hook provides access to current display mode. There are 4 display modes enabled in the editor by default. Some element settings are dependent on the display mode, and can be changed per display mode. Use this hook when you need to access the current display mode and its config (name, icon, etc.).

#### How to use this hook?

```typescript
import { useDisplayMode } from "@webiny/app-page-builder/editor/hooks/useDisplayMode";

const MyComponent = () => {
  const { displayMode, config } = useDisplayMode();
};
```

### useHighlightElement

This hook provides access to the element which is currently being hovered over (highlighted). Use this hook when you need to access the highlighted element and, optionally, highlight a different element.

#### How to use this hook?

```typescript
import { useHighlightElement } from "@webiny/app-page-builder/editor/hooks/useHighlightElement";

const MyComponent = () => {
  const [element, setHighlightedElement] = useHighlightElement();
};
```

### useUpdateElement

This hook is used to update the state of a given element. It also allows you to control whether the new state is stored into the state history, for undo/redo purposes.

#### How to use this hook?

```typescript
import { useUpdateElement } from "@webiny/app-page-builder/editor/hooks/useUpdateElement";
import { useActiveElement } from "@webiny/app-page-builder/editor/hooks/useActiveElement";

const MyComponent = () => {
  // Get element updater
  const updateElement = useUpdateElement();

  // Get an element you wish to update
  const element = useActiveElement();

  const onClick = () => {
    updateElement({ ...element, data: { ...element.data, custom: "data" } });
  };

  // Render your component
  return <button onClick={onClick}>Update active element!</button>;
};
```
